# Sales Performance Analysis Dashboard

## Project Overview
Interactive dashboard to monitor sales KPIs and 
business performance using Power BI and Excel.

## Tools Used
- Microsoft Excel (data cleaning)
- Power BI Desktop (dashboard and visualization)

## Dataset
Superstore Sales Dataset from Kaggle
9994 rows of sales transactions (2014-2017)

## Key Insights
- Total Sales: $2.30M
- Total Profit: $286.40K
- Technology is the highest selling category
- West region has the highest sales (31.58%)
- South region has the lowest sales (17.05%)

## Dashboard Features
- KPI Cards (Sales, Profit, Quantity)
- Sales trend line chart by Month-Year
- Sales by Category bar chart
- Sales by Region pie chart
- Region slicer for filtering
